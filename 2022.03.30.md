# 이산수학
3장 함수, 수열, 관계 단원의 소단원 3까지 공부함.

## 관계
+ 반사적 : (x, x)
+ 대칭적 : (x, y)일 때 (y, x)
+ 반대칭적 : 모든 쌍 사이에 하나의 간선만을 가짐.
  + 주의점! 대칭적이면서도 반대칭적일 수 있음. 서로 반대되는 개념이 아님!
+ 추이적 : (x, y), (y, z) 일 때 (x, z)
+ 반순서(부분 순서) : 트리와 비슷. 

## 느낀 점
+ 관계들이 좀 헷갈린다. 특히 반순서!
+ 확실히 이산수학이 컴퓨터의 기초 지식이 되는 것 같다.
